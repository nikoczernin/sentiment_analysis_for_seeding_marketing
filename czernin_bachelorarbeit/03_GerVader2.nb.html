Daten vorbereiten

sentencestidy

Es gibt 2,093,119 Stze im Datensatz. Diese werden in einem TSV gespeichert, um Bearbeitung mit GerVADER2 zu ermglichen.

GerVADER2 erwartet 3 Spalten: * id (nicht wichtig) * label (nicht wichtig) * comment_text (wird analysiert)

comments_for_TSV <- sentencestidy %>% 
  mutate(
    label  = paste(project_id, blog_id, comment_id, sep="_"),
  ) %>% 
  select(label, comment_text )
comments_for_TSV

# Speichere die Tidy-Text-Stze als TSV
trnd_sentences_tidy.path = "./data/trnd_sentences_tidy.tsv"
